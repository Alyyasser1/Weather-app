# Weather App

A simple, responsive web application that allows users to search for current weather conditions of any city worldwide using the Weatherstack API.

## Live Demo

Check out the live demo here: [https://alyyasser1.github.io/Weather-app/](https://alyyasser1.github.io/Weather-app/)

## Features

- **City Search**: Search weather information for any city in the world
- **Current Weather Display**: View real-time weather data including:
  - Temperature
  - Weather condition description
  - Humidity percentage
  - Wind speed
- **Dynamic Weather Icons**: Weather-appropriate SVG icons that change based on weather conditions
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Error Handling**: Displays a user-friendly message when a city is not found
- **Location Display**: Shows the searched city name and current date

## Technologies Used

### Frontend

- **HTML5**: Semantic markup for page structure
- **CSS3**: Styling and responsive layout with flexbox
- **JavaScript (ES6+)**: Dynamic content manipulation and API integration
  - Async/await for API calls
  - DOM manipulation with querySelector
  - Event listeners for user interactions

### APIs & External Services

- **Weatherstack API**: Real-time weather data provider
  - Free tier access for current weather conditions
  - Temperature in Celsius
  - Weather descriptions and codes
  - Humidity and wind speed data

### Assets

- **Font Awesome Icons**: For UI icons (search button, location marker, etc.)
- **Google Fonts**: Open Sans, Oswald, and Poppins fonts
- **Custom SVG Icons**: Weather condition icons (thunderstorm, drizzle, rain, snow, clear, clouds, atmosphere)

## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection
- Weatherstack API key (free tier available at https://weatherstack.com)
